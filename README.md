# sawk - some extras to POSIX awk

## Dependencies

* `gawk`
* `make`

## Installing

```terminal
$ make
# make install
```

## Usage

```terminal
# Format the file in place
$ awkfmt -w hello.awk

# Format to stdout
$ awkfmt hello.awk
```

More details on the [man page](https://github.com/cristianrz/sawk/blob/master/sawk.1.md).
